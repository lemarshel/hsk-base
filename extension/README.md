# HSK News Transcriber Extension

Captures the active tab audio and streams PCM to the backend for live subtitles.

## Install (Chrome)
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `extension/` folder

## Notes
- The news modal tab must be **active** for capture to work.
- If you use a different site URL, update `matches` in `manifest.json`.
